##### NixOS

A flake is provided that you can use to install the program on NixOS systems.
Optionally, you can also add the system package directly in your configuration.nix. To do so you will need to define the pixelfix input in the file parameters.

import flake from "./assets/flake.nix?raw";

export const flakefocus = [
	"recar",
	"git+https://codeberg.org/hamhim/recar/main",
	'recar.url = "github:hamhimdev/recar/main";',
	"recar.packages.x86_64-linux.default",
];

import configuration1 from "./assets/configuration1.nix?raw";
import configuration2 from "./assets/configuration2.nix?raw";
import configuration3 from "./assets/configuration3.nix?raw";

### Building from Source

To build Recar from source, you'll need `nodejs`, `pnpm`, and `git`.

<Steps>

1. **Clone the repository**:

    The `--recursive` flag is very important as [roverpp](https://github.com/TheUnium/roverpp/) (Recar Overlay), Equicord and Vencord are included as Git submodules.

    ```bash
    git clone --recursive https://codeberg.org/hamhim/recar.git
    cd recar
    ```

2. **Install dependencies**:

    ```bash
    pnpm install
    ```

3. **Build client mods**:

    ```bash
    pnpm build:mods
    ```

4. **Launch Recar**:
   `bash
    pnpm start
    `
</Steps>
