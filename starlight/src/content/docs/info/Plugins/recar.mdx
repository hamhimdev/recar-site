This plugin is a core plugin for a reason.  
It is responsible for connections between Recar and Discord. Additionally it makes some UI changes such as the Recar Settings button in Discord settings, and the Discord titlebar is made with this plugin. Our streaming implementation also almost fully relies on this plugin. It also exposes other metadata such as voice call updates for use in Recar Overlay. However, normal users cannot directly modify or configure this plugin, or turn it off easily due to its importance.
