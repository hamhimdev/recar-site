You can't do much with it if you're not developing for Recar. It's a core plugin for a reason.  
This plugin is responsible for connections between the UI of Recar and Discord, for example, the ringing popup system. Additionally it makes some UI changes such as the Recar Settings button in Discord settings, and the Discord titlebar is made with this plugin. Our streaming implementation also almost fully relies on this plugin. It also exposes other metadata for use in Recar Overlay for example.
